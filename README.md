Simple Shell Project
This project is a simple shell program implemented in C, which aims to replicate the functionality of a command-line interface. It provides a user-friendly interface for interacting with the operating system by accepting commands and producing corresponding outputs. It performs command parsing, execution, and handling of basic features, making it a practical tool for executing various system tasks.

Features
Command Execution: It executes both external programs and built-in commands.
Input/Output Redirection: Supports redirection of standard input and output, allowing data to be read from or written to files.
Background Processes: It shell can run processes in the background, enabling users to execute multiple tasks concurrently.
Built-in Commands: It shell includes built-in commands such as cd (change directory) and exit (terminate the shell).
Error Handling: It provides informative error messages and gracefully handles errors encountered during command execution.
